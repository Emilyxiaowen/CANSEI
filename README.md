## JT-OTA mock

スマートフォン操作を前提にした、JT-OTA向け高級インバウンド旅行アプリの静的モックです。

- 画面本文は日本語表示
- 言語切り替えは右上のシンプルなプルダウン
- 下部メニューバーでホーム、プラン、AIガイド、体験・購入、MyPageを切り替え
- AI会話型の旅程プランナー、英語名と外観デザイン付きのキャラクター選択式AIガイド、写真付き限定体験、お土産購入、MyPageのお客様情報登録を収録
- 富裕層ユーザー向けに、濃紺・ゴールド・シャンパンカラーを基調にした高級感あるトンマナ

`index.html` をブラウザで開くか、このディレクトリを静的ファイルサーバーで配信してください。
